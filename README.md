# 📊 Excel Data Analytics Projects

A collection of Excel-based data analytics projects, each demonstrating formula-driven analysis, 
pivot-style summary tables, and charts — no hardcoded values, everything calculated live from raw data.

## 📁 Projects

### 1. [Excel Module Assignment](./excel-module-assignment)
Core Excel skills exercise covering formulas, formatting, sorting/filtering, dates, logical functions, 
data cleaning, PivotTables, lookups, conditional aggregation, conditional formatting, and macros — 
built on a retail sales dataset.

### 2. [Hospital Readmission Analysis](./hospital-readmission-analysis)
Analysis of hospital readmission data to uncover patterns and risk factors behind patient readmissions, 
using Excel formulas, pivot tables, and charts to support data-driven healthcare decisions.

### 3. [Adecco HR Analytics](./adecco-hr-analytics)
HR analytics case study on employee attrition at a technology company, built on the IBM HR Analytics 
Employee Attrition dataset (1,470 employees). Answers 20 business questions on attrition drivers, 
job satisfaction, and retention strategy using live Excel formulas and charts.

**Key insight:** Overall attrition rate is 16.12%, with the Sales department showing the highest 
attrition at 20.63%.

## 🧰 Tools Used
- Microsoft Excel — formulas (COUNTIF, COUNTIFS, AVERAGEIF, CORREL), PivotTables, charts, trendlines
